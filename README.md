#Setup Environment

1. Install Lerna[]
2. Install Husky
3. Install [@commitlint](https://github.com/conventional-changelog/commitlint/tree/master/@commitlint/config-conventional)

* Commit available types:
[
  'build',
  'ci',
  'chore',
  'docs',
  'feat',
  'fix',
  'perf',
  'refactor',
  'revert',
  'style',
  'test'
];
